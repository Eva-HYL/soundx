# SoundX 产品总览

## 1. 项目定位

SoundX 是一个面向俱乐部/公会的陪玩 SaaS 平台，核心定位不是平台代收代付，而是提供订单协作、人工支付确认、积分结算、提现处理和俱乐部运营工具。

核心原则：
- 平台不接在线支付系统
- 平台不做资金托管
- 所有关键资金动作均通过人工转账和人工确认完成
- MVP 优先跑通闭环，不优先追求自动化和复杂撮合

## 2. 产品角色

### 用户端
- 微信登录
- 加入/切换俱乐部
- 浏览陪玩
- 指定下单或发布需求
- 查看订单
- 发起打赏

### 陪玩端
- 微信登录
- 加入俱乐部
- 抢单/接单
- 维护接单状态
- 提交服务报备
- 查看积分和积分流水
- 申请提现

### 俱乐部管理员端
- 确认订单付款
- 派单/处理订单池
- 审核服务报备
- 处理提现申请
- 管理陪玩与俱乐部配置
- 查看俱乐部统计

### 平台运营端
- 审核俱乐部入驻
- 管理套餐与功能开关
- 处理平台级异常与工单
- 查看平台总览数据

## 3. MVP 范围

### 必做
- 微信登录
- 用户切换俱乐部
- 陪玩资料与服务项展示
- 创建订单
- 管理员确认付款
- 抢单/指定单接单
- 服务完成后提交报备
- 管理员审核报备并发放积分
- 提现申请与人工转账确认
- 俱乐部后台待办处理

### 暂缓
- 在线支付接入
- 自动分账
- 复杂智能派单
- 广告商业化
- 大量增值服务能力
- 多端独立应用拆分

## 4. 核心业务闭环

### 订单闭环
用户下单 -> 管理员确认付款 -> 陪玩接单 -> 服务完成 -> 陪玩报备 -> 管理员审核 -> 积分到账

### 提现闭环
陪玩申请提现 -> 管理员审核 -> 管理员线下转账 -> 系统确认完成 -> 扣减积分

## 5. 产品边界

系统负责：
- 订单状态管理
- 人工确认留痕
- 报备审核
- 积分与提现记录
- 俱乐部 SaaS 管理能力

系统不负责：
- 在线收银
- 真实资金清结算
- 自动金融分账
- 第三方支付对账
